## 🎯 なぜMood Calendarなのか?

- **📊 気分を可視化** - カラフルなカレンダーで1ヶ月の感情の流れを一目で把握
- **💨 ストレス発散機能** - 嫌なことを書いて「飛ばす」!ロケット・爆発・シュレッダーの3種類のド派手エフェクトでカタルシス体験
- **🎯 目標管理** - 明日の目標を設定し、達成状況を追跡してモチベーション維持
- **🔮 毎日の占い** - 12星座の運勢ランキングで1日をスタート
- **🎨 5つのテーマ** - Light、MidNight、Future、Ruin、Crystalから選べる美しいUI
- **🔒 完全プライベート** - データはブラウザにのみ保存。ログイン不要、サーバー送信なし
- **📱 PWA対応** - インストールしてアプリのように使える

## ✨ 特徴

### 🌟 気分トラッキング
4段階の気分（😣 しんどい / 😐 ふつう / 🙂 わりと良い / 😄 最高）で毎日を記録。カレンダーが自動的に色分けされ、視覚的に気分の推移を把握できます。

### 💥 ネガティブ感情の浄化
嫌なことを書いて「飛ばす」ボタンを押すと...
- 🚀 **ロケット**: 嫌なことがロケットで宇宙へ飛んでいく
- 💥 **爆発**: 派手な爆発エフェクトで粉々に
- ✂️ **シュレッダー**: 切り刻んで消し去る

画面が揺れ、フラッシュエフェクトが走る、本格的な演出でスッキリ！

### 📈 分析機能
- **直近7日間のムードグラフ**: 気分の推移を棒グラフで表示
- **今月のムード分布**: 各気分の割合を可視化
- **平均ムードスコア**: 今月の気分を数値化
- **記録日数カウント**: モチベーション維持をサポート

### 🎯 目標設定機能
- 明日の目標を前日または当日に設定
- カレンダーに🎯マークで表示
- 達成状況を✅/❌で記録
- 目標達成の振り返り機能

### 🔮 占い機能
- 毎日更新される12星座の運勢ランキング
- ラッキーアイテムの表示
- 詳細な運勢メッセージ

### 🎨 テーマシステム
5つの美しいテーマから選択可能:
- **Light**: 爽やかな昼間の雰囲気
- **MidNight**: 静謐な深夜の森
- **Future**: サイバーパンクな未来都市
- **Ruin**: 神秘的な古代遺跡
- **Crystal**: 幻想的なクリスタル

各テーマには写実的な背景写真とSVGイラストが組み合わされています。

## 📱 基本操作

### 1. 気分を記録する
- カレンダーの**今日の日付**をクリック
- 4つの気分から1つ選択
- 良かったことをメモ（任意）
- 「Save」ボタンで保存

> ⚠️ **編集可能なのは今日の日付のみ**です。過去の記録は変更できません。

### 2. 嫌なことを飛ばす
- 右側の「嫌なことを飛ばす」セクションにテキストを入力
- 「💨 嫌なことを飛ばす」ボタンをクリック
- ランダムな派手エフェクトでストレス発散！

### 3. 目標を設定する
- トップの「明日の目標」セクションで「目標を設定」をクリック
- 明日の目標を入力（最大200文字）
- 翌日、「今日の振り返り」で達成状況を���録

### 4. 分析を見る
- 「分析」ボタンをクリック
- 直近7日間のムードグラフと今月の分布を確認

### 5. 占いをチェック
- 下部ナビゲーションの「🔮 占い」タブをタップ
- 今日の星座運勢ランキングを確認

## 🔧 カスタマイズ

### 占いデータの更新
```python
python update_fortune.py
```
`fortune.json`を編集して独自の占いコンテンツを追加できます。

### テーマの追加
`script.js`の`THEMES`配列と`THEME_PHOTOS`オブジェクトに新しいテーマを追加:

```javascript
{
  id: "custom",
  name: "Custom Theme",
  colors: ["#color1", "#color2", "#color3", "#color4"]
}
```
